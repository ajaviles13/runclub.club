# runclub.club

A clean landing page for the runclub.club domain. Visitors can reach out via Instagram.

## Setup

### 1. Install dependencies

```bash
npm install
```

### 2. Images

Drop your images into `public/images/`:

- `RunClubImage_1.jpeg` — Hero section background
- `RunClubImage_1.jpeg`, `RunClubImage_2.jpeg`, `RunClubImage_3.jpeg` — Gallery grid

## Local development

```bash
npm run dev
```

Open http://localhost:5173

## Build & deploy

```bash
npm run build
```

Deploy to Vercel:

```bash
vercel
```

Or connect the repo in the [Vercel dashboard](https://vercel.com/new).
